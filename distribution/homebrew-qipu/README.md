# homebrew-qipu

Homebrew tap for [qipu](https://github.com/mwaldstein/qipu).

## Installation

```bash
brew tap mwaldstein/qipu
brew install qipu
```

## Usage

After installation, `qipu` will be available in your PATH:

```bash
qipu --help
```

## Upgrading

```bash
brew upgrade qipu
```

## Uninstalling

```bash
brew uninstall qipu
brew untap mwaldstein/qipu
```

## About

This tap provides the `qipu` CLI tool, a knowledge graph CLI designed for scripts and agents.

For more information, visit the [qipu repository](https://github.com/mwaldstein/qipu).
